#  EduTech LMS on AWS ECS

Containerized and deployed an EduTech LMS on AWS ECS Fargate with a scalable, load-balanced architecture.

##  Key Features
- Dockerized frontend application
- Deployed on AWS ECS Fargate
- Application Load Balancer for traffic distribution
- Secure networking with Security Groups

##  Troubleshooting Experience
- Fixed container configuration issues
- Resolved ALB health check failures
- Debugged networking & security group issues

## Tech Stack
- AWS ECS (Fargate)
- Docker
- Application Load Balancer
- Git & GitHub

##  Outcome
Successfully deployed a production-ready, scalable LMS and gained hands-on experience in cloud support and DevOps practices.
